Namespace définissant une fonction manquante dans le namespace {\bfseries std} . Le namespace {\bfseries patch} crée un template définissant un fonction qui n'est pas incluse dans le namespace {\bfseries std}. Cette fonction {\bfseries to\-\_\-string} redéfinie le type d'une variable {\itshape n} quelconque.
