\subsection{Detailed Description}
Définition des messages d'arrêt de la communication client-\/serveur. Ce fichier contient les fonctions d'affichage d'un message d'erreur destiné à l'utilisateur. Cette erreur survient lorsque la connection n'arrive pas à être établie entre le serveur et le client. Les fonctions quittent également le programme après que l'affichage de l'erreur ait été terminé.
